# Wingmatch
A dating website project from Team Brogrammers for WS2025 Software Engineering.
Team Members:
  Coordinator: Alfonso Joaquin De Vera
  Lead Developers: Loretta Malinovic, Alexandra Monte
  Lead Designer: Joel Karakkattu

Wingmatch is a dating website where people can meet and match with others based on preferences, with the added bonus of having "Wingmen" commenting on your profile.
